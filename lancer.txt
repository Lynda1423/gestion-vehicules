=== ARRÊT PROPRE ===
1. Arrêter le frontend : 
   Fais Ctrl + C dans la fenêtre PowerShell où tourne le frontend.
2. Arrêter Kubernetes : 
   Ferme simplement Docker Desktop (ou éteins ton PC). Kubernetes mettra tout en pause automatiquement. Tes bases de données seront conservées.

=== LANCEMENT PROPRE ===
1. Démarrer Kubernetes : 
   Ouvre Docker Desktop et attends quelques minutes que tes pods (sgfv) soient tous relancés automatiquement.
2. Connecter l'API (Port-forward) : 
   Ouvre un terminal et tape la commande suivante :
   kubectl port-forward -n sgfv deploy/api-gateway 4000:4000
3. Lancer le frontend : 
   Ouvre un nouveau terminal et tape :
   .\start-frontend.ps1
4. Accéder à l'application : 
   Va sur ton navigateur à l'adresse http://localhost:3005
